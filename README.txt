# Star Locksmith Las Vegas LLC Website

Simple static website ready for GitHub Pages.

## Files
- `index.html`
- `style.css`

## How to publish on GitHub Pages

1. Go to https://github.com
2. Create a new repository
3. Name it something like:
   `star-locksmith-las-vegas`
4. Upload these files:
   - `index.html`
   - `style.css`
5. Go to:
   Settings → Pages
6. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
7. Click Save

Your website will be published at a GitHub Pages link like:

https://yourusername.github.io/star-locksmith-las-vegas/
